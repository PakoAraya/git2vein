### Bloques de código

Los bloques de código se hacen mediante ` ``` `
Al inicio de cada bloque de código se debe poner el lenguaje de programación.
Por ejemplo ` ```java `
Luego se debe poner el código dentro de las comillas.
Por ejemplo ` ```java
public static void main(String[] args) {
System.out.println("Hello World");
}
Y se vera así...

```java
public static void main(String[] args) {
    System.out.println("Hello World");
}
```
